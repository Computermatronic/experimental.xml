    Module std.experimental.xml.lexers

    This module implements various XML lexers. The methods a lexer should implement are documented in experimental.xml.interfaces.

    Structs

    Name Description
    RangeLexer A lexer that takes an InputRange.
    SliceLexer A lexer that takes a sliceable input. It doesn't really need an allocator. It takes it to have the same parameters as the others.
