    Module std.experimental.xml.interfaces

    This module contains some traits that serve as interfaces for some components of the library.

    Structs

    Name Description
    XMLToken The structure returned in output from the low level parser. Represents an XML token, delimited by specific patterns, based on its kind. This delimiters shall not be omitted from the content field.
