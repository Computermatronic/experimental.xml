    Enum member isLowLevelParser

    Checks whether its argument fulfills all requirements to be used as XML lexer.

    An XML lexer is the first component in the parsing chain. It masks from the parser the shape of the input and the type of the characters in it. It must be an InputRange of some instantiation of LowLevelNode.

    Declaration

    enum isLowLevelParser(P) = isInputRange!P && is(ElementType!P : XMLToken!(P.CharacterType)) && is(typeof((inout int = 0) { alias InputType = P.InputType; alias CharacterType = P.CharacterType; P parser; InputType input; parser.setSource(input); } ));
