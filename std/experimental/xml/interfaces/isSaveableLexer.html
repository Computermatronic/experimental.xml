    Enum member isSaveableLexer

    Checks whether the given lexer is savable.

    The method save should return an exact copy of the lexer that can be advanced independently of the original.

    Declaration

    enum isSaveableLexer(L) = isLexer!L && is(typeof((inout int = 0) { const L lexer1; L lexer2 = lexer1.save(); } ));
