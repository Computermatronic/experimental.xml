    Struct XMLToken

    The structure returned in output from the low level parser. Represents an XML token, delimited by specific patterns, based on its kind. This delimiters shall not be omitted from the content field.

    Template arguments

    struct XMLToken(T);

    Fields

    Name Type Description
    content T[] The content of the token
    kind XMLKind Represents the kind of token
