    Enum UserDataOperation

    An integer indicating the type of operation being performed on a node.

    The enum base type is ushort .

    Enum members

    Name Description
    NODE_ADOPTED The node is adopted, using Document.adoptNode().
    NODE_CLONED The node is cloned, using Node.cloneNode().
    NODE_DELETED The node is deleted.

    Note

    This may not be supported or may not be reliable in certain environments, where the implementation has no real control over when objects are actually deleted.

    NODE_IMPORTED The node is imported, using Document.importNode().
    NODE_RENAMED The node is renamed, using Document.renameNode().
