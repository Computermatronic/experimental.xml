    Function chooseLexer

    Instantiates a SliceLexer specialized for the given input

    Prototype

    auto chooseLexer(alias Input)();

    Parameters

    NameDescription
    Input a type for which to instantiate a SliceLexer, or a value for whose type to instantiate a SliceLexer

    Returns

    An instance of SliceLexer specialized for the given Input. Note that even if Input is a value, setSource is not called on the returned lexer

    Template chooseLexer

    Instantiates a lexer specialized for the given input and allocator

    Arguments

    template chooseLexer(alias Input, Alloc, Options...);

    Returns

    An instance of most suitable lexer type specialized for the given Input and Alloc. Note that even if Input is a value, setSource is not called on the returned lexer
